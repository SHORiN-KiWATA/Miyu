1. 安装
安装`fcitx5-im`作为基础的输入法框架，然后安装自己喜欢的输入方案。有几个推荐的方案：1. `fcitx5-chinese-addons`是一个合集包，包含了大多数常用的中文输入法方案，缺点是词库不太行，所以更推荐使用基于rime输入法引擎的输入方案。2. `fcitx5-rime rime-ice-git`是rime输入法引擎和雾凇拼音（这个包在aur上），除了雾凇拼音，另一个推荐的基于rime的输入方案是`rime-wanxiang-pinyin`万象拼音。

2. 配置
装完软件包之后需要编辑`~/.local/share/fcitx5/rime/default.custom.yaml`启用输入法，以下是推荐设置，每一个schema都代表一个输入方案，可以自选要启用的方案：
```
patch:
  schema_list:
    - schema: luna_pinyin_simp
    - schema: rime_ice
    - schema: wanxiang
    - schema: double_pinyin_flypy
    - schema: wubi86
    - schema: bopomofo
```

然后编辑环境变量

   - GNOME的话写入

     ```
     XIM="fcitx"
     GTK_IM_MODULE=fcitx
     QT_IM_MODULE=fcitx
     XMODIFIERS=@im=fcitx
     ```

     如果出现吞字问题可以尝试用`XDG_CURRENT_DESKTOP=GNOME`环境变量解决

   - KDE/wayland合成器的话写入

     ```
     XMODIFIERS=@im=fcitx
     ```

   - wayland合成器
      
      在配置文件里设置环境变量，例如：

      - niri

        ```
        environment {
              XMODIFIERS "@im=fcitx"
        }
        ```

      - hyprland

        ```
        env = XMODIFIERS,@im=fcitx
        ```

3. 最后调整系统设置

   - GNOME的话

    商店搜索extension，安装蓝色的extensionmanager，然后在扩展商店安装[input method panel](https://extensions.gnome.org/extension/261/kimpanel/)扩展。

   - KDE的话

     1. 打开`系统设置` > `键盘` > `虚拟键盘`， 选择`fcitx5 wayland启动器`，记得`应用`。
     2. `系统设置` > `语言和时间` > `输入法`，添加`rime（中州韵）`。 `配置全局选项`可以设置切换输入法的快捷键。`配置附加选项`可以进行一些自定义设置。

   - wayland合成器的话

     1. 打开`fcitx5配置`进行设置

     2. 在合成器配置文件里设置自动启动fcitx5

4. 出现异常的解决办法

    如果出现漏字问题可以设置LC_CTYPE环境变量为英文。如果软件完全无法使用输入法可以在启动时尝试加上`env GTK_IM_MODULE=fcitx QT_IM_MODULE=fcitx`，如果是electron应用的话加上`--enable-features=UseOzonePlatform --ozone-platform=wayland --enable-wayland-ime`
